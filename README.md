# DK8S -> Descomplicando Kubernets (treinamento da LinuxTips)
Este repositório foi criado como parte de um treinamento em Kubernetes (K8s). Seu propósito é registrar anotações, lembretes e aprendizados adquiridos ao longo do estudo, servindo como um material de consulta rápida e também como fonte de conhecimento para outras pessoas interessadas em aprender mais sobre Kubernetes.

A ideia é manter um espaço organizado, acessível e colaborativo, onde conteúdos importantes sobre Kubernetes possam ser revisados, ampliados e compartilhados.
